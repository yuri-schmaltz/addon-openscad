# Tests

Este addon possui um parser/evaluator proprio. Para validar rapidamente:

1. Abra um bloco de texto no Blender com script OpenSCAD.
2. Use o painel OpenSCAD Bridge > Preview.
3. Verifique se objetos sao criados na colecao OpenSCAD Preview.

Testes automatizados podem ser executados com o Blender em modo headless usando `--python` para scripts de teste.
